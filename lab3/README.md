    op[2]   : 若为1，则 ALU 输出桶形移位器的结果
    op[3]   : 若为1，则 ALU 输出加法器的结果
    op[4]   : 指示 is_unsigned 信号，1表示为无符号数，0表示为有符号数
    op[5]   : 指示 is_sub 信号，1表示做减法，0表示做加法
    op[6]   : 若为1，则 ALU 输出相与的结果
    op[7]   : 若为1，则 ALU 输出相或的结果
    op[8]   : 若为1，则 ALU 输出相异或的结果
